Папка приложения находится в папке проекта, при необходимости можно найти после компиляции в employeeAcc/bin/debug.
Проект запускается через visual studio.
Изменение значения происходит в самой таблице по двойному щелчку.
По окончании редактирования необходимо сохранить изменения либо по правой кнопке мыши либо по кнопке Сохранить изменения.
Для увольнения сотрудника необходимо выбрать сотрудника и правой кнопкой нажать удалить.
Для сохранения xml файла имеется кнопка в меню Файл.
Для поиска предусмотрено поле ввода, работает при нахождении первого совпадения с любым текстом в поле ФИО.
